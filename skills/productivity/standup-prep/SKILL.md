## Step 5 ‚Äî Write morning brief to file

Write the morning brief from Step 4 to a numbered file inside a dated subfolder:

```
{{AI_ROOT}}\Agents\Morning Briefs\YYYY-MM-DD\NN.md
```

**Rule ‚Äî always use the subfolder, never a flat file:**

1. Check `{{AI_ROOT}}\Agents\Morning Briefs\YYYY-MM-DD\` for existing files.
2. Write to the next available number: `01.md` if the folder is empty or doesn't exist, `02.md` if `01.md` exists, `03.md` if `02.md` exists, etc.
3. Create the subfolder if it doesn't exist.
4. **Never overwrite an existing file. Never append to an existing file.**

File format:

```md
# Standup ‚Äî YYYY-MM-DD / Standup N

<full morning brief content from Step 4>

---

_Generated by /standup-prep ¬∑ Model: <model name> ¬∑ Session: <session ID>_
```

Determine the actual output path before writing, then confirm: `‚úÖ Morning brief written to {{AI_ROOT}}\Agents\Morning Briefs\YYYY-MM-DD\NN.md`

After writing the file, also sync it to Asana automatically ‚Äî no prompt needed:

```sh
cd {{PROJECT_ROOT}} && npx tsx scripts/sync-morning-briefs.ts
```

This creates one task in the **Morning Briefs** Asana project (`morningBriefsProjectGid` in `.asana-config.json`) with the full brief content as the task notes. The script is idempotent ‚Äî if a task for this brief already exists it is skipped.

`üìã Brief synced to Asana Morning Briefs project`
